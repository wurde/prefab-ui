# Hello

[<- Back](../../README.md)

A simple 'hello world' greeting.

## Getting Started

Install the package:

```bash
npm i pf-bar-chart
```

Import the component:

```javascript
// ./src/index.js
import './web-components';

// ./src/web-components/index.js
import 'pf-hello';
```

Use the component:

```html
<pf-hello></pf-hello>
```
