# Prefab Bar Chart

[<- Back](../../README.md)

A simple Bar Chart.

## Getting Started

Install the package:

```bash
npm i pf-bar-chart
```

Import the component:

```javascript
// ./src/index.js
import './web-components';

// ./src/web-components/index.js
import 'pf-bar-chart';
```

Use the component:

```html
<pf-bar-chart></pf-bar-chart>
```
